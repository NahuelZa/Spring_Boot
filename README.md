# Spring_Boot - Mi Primer Proyecto 🚀

¡Bienvenido! Este es mi primer proyecto desarrollado con **Spring Boot**. El objetivo de este repositorio es documentar mi aprendizaje sobre el desarrollo backend con Java, el ecosistema de Spring y la persistencia de datos.

## 🛠️ Tecnologías Utilizadas

* **Java 17** (o la versión que estés usando)
* **Spring Boot 3**
* **Spring Data JPA** - Para la gestión de la base de datos.
* **Maven** - Gestión de dependencias.
* **H2 / MySQL / MongoDB** - (Elige la que estés usando actualmente).

## 📋 Requisitos Previos

Para ejecutar este proyecto localmente, asegúrate de tener instalado:
* JDK 17 o superior.
* Tu IDE favorito (IntelliJ IDEA, Eclipse o VS Code).
* Maven (opcional, puedes usar el `mvnw` incluido).
